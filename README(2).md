# Calculator

## Overview
A modern calculator built with HTML, CSS, and JavaScript featuring a vintage-inspired interface and responsive button layout.

## Features
- Basic arithmetic operations (+, −, ×, ÷)
- Percentage and sign toggle
- Decimal number support
- Keyboard input support
- Clear function
- Responsive calculator display
- Styled UI with animated button feedback

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## How to Run
1. Download the project.
2. Open `calculator.html` in any modern web browser.
3. Start performing calculations.

## Keyboard Shortcuts
- Numbers: 0–9
- Operators: + - * /
- Enter or = : Calculate
- Backspace: Delete last digit
- Escape: Clear
- %: Percentage
